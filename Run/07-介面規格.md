# 07 — 下班主畫面（給設計）

每天下班後跳出來的主畫面：你坐下來，決定今晚要把自己交給誰、交給什麼。

## 氣氛
- 一個普通人下班、回到一個人住的租屋的**晚上**。
- 進門前留的**那盞燈**還亮著；窗外城市（信義那側）很亮、很遠，屋裡卻很安靜。
- 低彩度、偏暗，像**深夜一個人滑手機**那種感覺——私密、有點空，但**不悲情**（隱形但不孤單）。
- 前期偏**堤防的安靜**；越往後，**信義的光與吵**越滲進畫面。← 氣氛本身就是進度。

## 畫面要有的東西
1. **狀態列**：左上 `Day X`＋進度（第N週／跑團人數）；右上 **今晚精力**（幾格）、**錢**、**設定**。
2. **背景**：堤防↔信義夜景＋主角背影（＝你）。隨進度從堤防偏向信義。
3. **今晚行動區**（核心）：一排卡，搶你那幾格精力——
   - 一般行動：去堤防跑／去信義跑／自主加班／開手機／早點睡。
   - **約定卡**：跟誰約了什麼（頭像＋事由）＋「執行」按鈕；不去就過期。
   - 每張卡＝圖示＋名稱＋**一句感覺提示（不是數字）**＋精力消耗。
4. **手機／社交面板**：限時動態、通知（誰看了你的限動／已讀不回／傳訊息）、交友軟體、「分享到你的限動」。**關係全靠這裡感覺，沒有數字。**
5. **系統入口**：圖鑑、存檔／讀檔、設定。

## 鐵律
**看得到＝精力、錢；看不到＝關係。** 絕不放好感度條、愛心、血條。
