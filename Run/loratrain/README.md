# 開局角色 LoRA 訓練集

這個資料夾收錄第一章開場小團的六名核心角色：`ray`、`kevin`、`yijun`、`laowu`、`ann`、`jason`。

## 目錄

每名角色使用相同結構：

- `reference/`：從遊戲現有定稿圖複製出的身份與畫風錨點；原本 `art-refs/` 不會被修改。
- `dataset/`：可直接交給支援同名 `.txt` caption 的 LoRA 訓練器；每張透明 PNG 都有同名 caption。`000*` 是定稿錨點的訓練用副本，其餘是新生成變體。
- `PROMPTS.md`：角色固定特徵、生成規則與每張訓練圖的鏡位規格。
- `CHARACTER.md`：從企劃文件複製的人物側寫與美術設定；總索引見 `CHARACTERS.md`。
- `_work/`：灰底測試與綠幕中間檔，已由本資料夾的 `.gitignore` 排除，不應拿去訓練。

## 訓練原則

- Trigger token 固定放在 caption 第一個詞，避免和一般人名混淆。
- 臉型、五官、髮型、年齡感、身材比例與招牌細節固定。
- 景別、角度、表情、姿勢和服裝要分散，避免 LoRA 只學會一張立繪。
- 服裝保留角色色彩語彙，但不可每張完全相同，降低服裝與身份綁死的風險。
- 每張只出現該角色一人；不要文字、浮水印、邊框、拼貼或角色設定表。
- 生成圖先人工淘汰錯手、錯臉、年齡漂移與配件漂移，再進訓練。

這批可作第一輪小型 LoRA；正式定稿前仍建議每角擴到 15–20 張合格圖。資料寧可少而準，不要把錯臉餵進 LoRA。

注意：所有 PNG 都保留 alpha。訓練器若不支援 alpha，請先明確指定合成底色；不要讓工具默默全部壓成黑底，否則 LoRA 可能把黑底一起學進去。
