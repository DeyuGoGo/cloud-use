## 人物側寫

**一句話**：他這輩子最會被人喜歡，學會的卻不是怎麼選一個，是怎麼一個都不讓他失望——等他真的只想對一個人好，才發現自己連「對不起，我有喜歡的人了」這句話都沒講過、不會講。

- **表**：團草。長得好、笑起來眼睛會彎，講話軟、不油，誰湊過來他都接得住，從不擺臉色、從不句點人。約他十次他答應十次，臨時改地點他說好、有人多揪一攤他也去，散場永遠最後一個走——總有人還想多講兩句，他不忍心先抽身。在這個人人都缺人看的場子，「靠近凱文不會被拒絕」是最稀有的溫柔，所以大家都想靠近他。看起來游刃有餘、八面玲瓏。
- **裡**：那份八面玲瓏是怕，不是爽。他不是在經營誰，是一個都不敢得罪——更準是，受不了任何一張臉在他面前垮下來。「拒絕」兩個字一到嘴邊，他眼前就先浮出對方被推開那一秒的表情，他擋不住，只好說好。被一堆人圍著時他其實是累的、分身乏術的；他不是不想專一，是從沒人教過他，討好每個人和對一個人認真，是兩件相反的事。
- **想要**：「我就喜歡跟人好好相處啊，能讓人開心不好嗎？」——他要的是每一張臉在他面前都是笑的、沒有一個人因為他而失望。這版本他真心相信，也真以為自己只是人緣好、心軟，不是在惹麻煩。
- **為了什麼（傷口·作者用，不明寫，絕不演出原生家庭場面）**：他真正缺的，是一次「我讓某個人失望了、對方也沒有就此不要我」的經驗——好讓他敢說一次不。他很早就學會「只要我夠好、夠順，沒人會生我的氣、沒人會走」，把「被喜歡」跟「我是安全的」綁死了。所以他不是貪心，是怕——拒絕一個人＝賭對方會不會因此不要他，這個賭他輸怕了，乾脆永遠不賭、永遠說好。他從沒學過：有些人是你拒絕了、反而更信你。
- **內耗**：分兩段、都不靠他自我診斷。其一（緋聞怎麼長出來）：那個對他特別好、天天傳訊息的女生，他心裡清楚自己沒那麼喜歡，可每次想說清楚，打開對話框、看到對方上一句還在期待，手指就僵在那——最後傳出去的還是個笑臉貼圖，又拖一天。他不是在吊她，是真的狠不下心讓她當場難看；可這份狠不下心，在她和旁人眼裡，就是「他也對我有意思」。其二（更狠，他第一次想專一）：遇到那個他真的喜歡、卻因為聽過他緋聞而躲他的人，他生平第一次想去跟其他人把話講清楚「我們只是朋友」——訊息打完又刪、刪完又打，排到第七遍，怕的還是那同一張會垮下來的臉。他這輩子最會搭話，唯獨「對不起，我心裡有別人了」這句，他從沒講過，發現自己根本不會講。
- **破綻**：要說不的那一刻，他的視線會先從對方臉上飄開——飄向地上、手機、旁邊任何一個人——然後那句「不」就變成「啊……再說啦」「改天喔」。配套物件：手機裡好幾條已讀、卻停在「正在輸入…」打了又刪的回覆。
- **怎麼不明講地演**：場一（＝序章第10場）：紅燈，他跟你並排聊得正暖，旁邊有人插話要拉他去前面，他「喔好、來了」應著、視線卻沒離開你，臉上是「不好意思我也不想走」的為難，笑沒斷、人卻被半推著拖走了——玩家讀到的不是「他在物色下一個」，是「他誰都拒絕不了」。場二（緋聞現場）：那個天天傳訊息的女生當眾挽他手「我們昨天還一起吃飯」，凱文沒甩開也沒承認，只是視線飄開、笑得有點僵「呃對啊……」——他越不忍心拆穿，旁人越坐實那層曖昧。場三（他第一次想推開人）：穩定追他的女生鼓起勇氣約他單獨出去，凱文破天荒想說不——他手指在螢幕上「我那天……」打了三個字，停住，看一眼對方滿臉期待的限動，刪掉，最後傳出去的是「好啊那約一下」，傳完對著手機嘆了一口很長的氣。明明想拒絕，手卻又答應了。

> **緋聞線（怎麼長出來、她怎麼因此不敢靠近——全用事件，不旁白）**：起點都是「不忍心」——新人問堤防路線，他不只回、還說「不然我帶你跑一次」（他對誰都這樣，對方卻截圖發限動「凱文要帶我跑欸」）；天天傳訊息的女生他不忍心已讀不回、每次回個貼圖，被截圖「他都會回我耶」；散場他總最後走、誰想聊他都奉陪。三件單看都只是心軟，湊在一起就有了形狀。傳開：超商攤有人半開玩笑「凱文你到底跟幾個在曖昧？」全桌笑，他也只能跟著笑、沒否認（否認＝當眾打誰的臉，他做不到）——「中央空調」的名號就這樣坐實、從這攤傳到下一攤。**他喜歡的人怎麼因此不敢靠近**：她原本願意跟他多講兩句，某天聽見隔壁桌兩個女生對帳「他也約過我」「我也是欸」，當場把到嘴邊的話收回去；後來凱文難得主動傳訊息、語氣明顯跟對別人不一樣，她盯著看很久，回了句淡淡的「你不是對每個人都這樣？」就把對話關掉——不是她不心動，是她沒辦法賭。最毒的一筆：凱文想證明這次不一樣，可他能拿出來的「證據」全是他過去對誰都好的同一套（記得你提過的事、散場等你、傳訊息關心）——他越想證明專一，看起來越像又一次中央空調。先前的好心，全變成了不利於他的證詞。

> **製作備忘**：① **跟 Ray 區隔**——Ray 主動經營魅力、為了當「全場中心的人物」；凱文被動、軟、根本沒在經營，麻煩是送上門他擋不掉：一個的魅力是武器，一個的魅力是他卸不下的負擔。② 這版**把凱文移出「付出換留下」家族**（原本跟 Ray／Ann 撞引擎）——他的恐懼是「不忍心讓任何人在我面前失望」，獨一份。③ 「中央空調」是**別人給的名聲**、不是他的真面目（裡頭只是軟），這個落差就是他的悲劇。④ 全篇只留一處「他這輩子最會被人喜歡」（在一句話），其餘交給事件、不替玩家反覆總結。⑤ 序章 Opening.gd 場10 已同步改成這版（被叫走時的為難，不是掃全場偵查）。⑥ **關係定錨**：凱文↔怡君＝曖昧緋聞＋**肉體關係**，但**不是情侶**——怡君只是來紓壓，看穿他中央空調也不排斥、頂多抱怨兩句（見怡君）。⑦ 凱文**真心喜歡、卻因名聲不敢靠近他的「那個人」另有其人（待定，非怡君）**——緋聞線裡那個「不敢賭」的她＝這個待定對象，不要寫成怡君。⑧ R18：情慾與曖昧維持暗示／淡出，不寫露骨（依 README 鐵律）。

## 美術設定

- **一句話**：跟他講話，周圍好像自動消音——他讓你覺得「全場只有你重要」。大家要他不是因為帥，是靠近他有「被選中」的感覺。
- **外型**：陽光、好親近的運動型，**不是高冷帥哥**；笑起來眼睛會彎。
- **招牌**：**自然、不費力**的運動短打（刻意和 Ray 的「精心打扮」相反）；放鬆的肢體。
- **神情姿態**：身體微微傾向你、專注的眼神、像真的覺得你有趣的笑——這份「注意力」就是他的武器。
- **破綻**（細微，別太明顯）：那份笑是他**給每個人**的反射動作；可在眼神最末端藏一絲已經要飄向下一個人的鬆動。但主視覺仍要讀成「真誠的暖」。
- **別畫成**：渣男、輕浮、把妹臉。他不油——**他像真的**，那才是陷阱。
