# 《約跑團》專案文件 ＋ HANDOFF

> 18+ 心理向手機敘事遊戲。取材自 **2026 台北信義區「社交路跑」炎上事件**。
> 一句話：一個被忽略的上班族下班來跑步紓壓，在一個「被看見才算數」的地方，靠著一連串「完全正確」的選擇把自己擺第一，最後變成他最不想變成的人——而他問心無愧。

## 先讀這個（給接手的人／另一個 AI）
**最重要的是 [01](01-世界觀與主線.md) 裡的三塊，讀完就抓得到靈魂：**
1. **核心引擎：正當的自利** — 一個看似正確、維護自己權益的選擇，傷了別人的感受，然後漸行漸遠。沒有壞事、沒有那一刀，加害者找不到自己哪裡做錯。
2. **基調備忘（鐵律）** — 不要 melodrama、不要背叛事件；寫「溫度怎麼掉的」，不要寫「他做了什麼壞事」。
3. **寫作範例（老周那杯沒帶的咖啡）** — 整個腳本要寫到那個密度。
